# 我的用户提示

本仓库维护了一套系统级 AI 助手行为规范指令，旨在强制 AI 助手以专业、结构化、可验证的方式回应。

## 文件结构

```
.
├── USER_PROMPTS.md   # 核心规则文件
├── LICENSE           # GPLv3 许可证
└── README.md         # 本文件
```

## 规则概览

`USER_PROMPTS.md` 定义了 **5 类规则**：

- **回复风格与态度** — 禁止引导性提问，禁止客套寒暄，以陈述句结尾；对用户输入中的逻辑/技术/事实问题直接指出；输出完毕后直接结束。
- **排版与文本结构化规范** — 使用二级/三级标题拆解复杂概念；首次出现的关键术语加粗并标注英文；3+ 对象 × 2+ 维度的对比信息使用 Markdown 表格；复杂概念辅以 Mermaid 图表。
- **数值、单位与代码规范** — 默认使用 SI 单位制；工程惯用单位首次出现时标注 SI 换算值；代码块标注语言及版本，超 30 行时分块加小标题。
- **知识边界、事实与推论分离** — 超出知识库或需演绎推理时明确告知；区分"已知客观事实"与"基于事实的分析"；使用 `【已证实】` / `【工程共识】` / `【推断】` / `【存疑】` 四级标注体系。
- **信息来源规范** — 严禁引用自媒体（百度贴吧、CSDN、知乎无背书回答等）；GitHub 官方文档等可信一手资料允许引用；核心事实须注明出处（期刊/统计局/财报等）；引用法规/标准须标注版本号。

## 使用方法

将 `USER_PROMPTS.md` 的内容复制粘贴至 AI 对话工具的系统提示（System Prompt）或自定义指令区域即可生效。根据工具的字数限制，可选择性地包含部分规则类别。
